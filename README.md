## Using This Wiki

### On GitHub

This wiki is designed to be used with GitHub Wiki. To enable:

1. Fork/clone this repository
2. Go to your repo's Settings → Features → Enable Wikis
3. Clone the wiki: `git clone https://github.com/YOUR_USERNAME/spoolbuddy-wiki.wiki.git`
4. Copy the markdown files to the wiki repository
5. Push to see the wiki on GitHub

### Locally

You can view these markdown files directly or use a markdown viewer.
